# Website Khurusus Java Teory (Dasar)
Situs ini diperuntukan untuk edukasi, 
Memahami teori-teori umum dalam bahasa pemrograman java.
dibuat dalam one Page HTML,
Dengan banyak variasi Assesment untuk menguji pemahaman kamu.
Situs ini terdiri dari 14 modul, dengan chalange 7x pengulangan,
Minimal kelulusan 70%++ untuk Lulus.
dan di Akhir materi kamu dapat mencetak sertifikat untuk tanda kerberhasilanmu.
